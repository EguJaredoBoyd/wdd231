My Programming Journey - Site Plan

Site Name

Name: My Programming Journey

This website will serve as a personal blog documenting my progress as a beginner programmer aspiring to become a professional developer.

Site Purpose

The website will track my personal progress in web development and coding. It will also serve as a portfolio to showcase my projects and as a resource hub for beginners interested in programming. As I grow in my coding skills, this site will evolve with me, highlighting my milestones and achievements.

Scenarios

  • What steps did I take to learn programming?
  • Where can I find coding projects and resources for beginners?

Secondary Color: #ECF0F1 (Used for backgrounds and accents)

Typography
